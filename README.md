# zylos-dashboard-builder

Reusable dashboard builder skill for 3AI agents.

## Purpose

This repository contains a public-safe dashboard component library and page-building guide. It is intended for creating static HTML dashboards from structured data and page config.

Primary skill entry:

```text
SKILL.md
```

## Public-Safe Rules

- Do not commit customer data, internal deployment paths, tokens, credentials, or private dashboard exports.
- Examples must use synthetic or clearly public data.
- Generated dashboards intended for public sharing must pass a public-safe review before publication.
- Private operational details belong in private 3AI repositories, not in this public repository.
